# 条件执行

if、else 和 elif 语句控制条件代码执行。条件语句的语法格式为：

```python
if expression:
    statements
elif expression:
    statements
elif expression:
    statements
...
else:
    statements
```

如果不采取任何操作，则可以省略条件语句的 else 和 elif 子句。如果特定子句不存在任何语句，使用 pass 语句：

```python
if expression:
    pass # To do: please implement
else:
    statements
```
