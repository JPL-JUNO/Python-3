# 映射操作

映射是键和值之间的关联。内置的 `dict` 类型就是一个例子。下表中的操作可以应用于映射。

|Operation|Description
|---|---
|`x = m[k]`|通过键来索引
|`m[k] = x`|通过键来赋值
|`del m[k]`|通过键来删除一项
|`k in m`|成员检查
|`len(m)`|映射的项目数
|`m.keys()`|返回键的视图
|`m.values()`|返回值的视图
|`m.items()`|返回键值对的视图

键值可以是任何不可变的对象，例如字符串、数字和元组。当使用元组作为键，可以省略括号并写入逗号分隔的值：

```python
>>> d = {}
>>> d[1, 2, 3] = "foo"
>>> d[1, 0, 3] = "bar"
>>> d
{(1, 2, 3): 'foo', (1, 0, 3): 'bar'}
>>>
```

**使用元组作为键是创建组合键的常用技术。**
