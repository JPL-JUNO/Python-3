>>> a = 37      # 用值 37 创建一个对象
>>> b = a       # 对 37 增加引用计数
>>> c = []      
>>> c.append(b) # 对 37 增加引用计数

一个对象的引用计数在遇到 del 语句、引用超出返回或引用重新分配时会减少

>>> del a      # 对 37 减少引用计数
>>> b = 42     # 对 37 减少引用计数
>>> c[0] = 2.0 # 对 37 减少引用计数

对象的当前引用计数可以通过 sys.getrefcount()函数获得

>>> a = 37
>>> import sys
>>> sys.getrefcount(a)
1000000007
